# Resume Job Matcher

A full-stack web application where users upload resumes and get job recommendations
based on skills and keywords.

## Tech Stack
- FastAPI
- React + Tailwind CSS
- spaCy
- SQLite

## Available Scripts

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
